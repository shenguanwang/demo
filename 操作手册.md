# 华为系新能源汽车海外获客工作台操作手册

更新日期：2026-07-16
适用版本：新加坡服务器正式版

## 一、系统入口

### 1. 官网

```text
https://www.yiming-auto.com
```

根域名 `https://yiming-auto.com` 会自动跳转到 `www` 官网。

### 2. CRM 工作台

```text
https://crm.yiming-auto.com/login.html
```

工作台只能通过 CRM 域名进入。公网 IP 入口已关闭，不要使用服务器 IP 登录。

### 3. 账号说明

- 管理员：可以管理用户、负责国家、系统设置、API、任务和业务数据。
- 普通用户：只能查看和操作自己的线索、客户、报价及任务。
- 每个账号的数据相互隔离。
- 普通用户只能搜索管理员为其分配的国外国家。
- 负责国家设为“无”时，该用户不能使用“一键获客”。
- 国内区域不受国外负责国家限制。

不要在操作手册中保存账号密码或 API Key。

## 二、标准业务流程

```text
市场选择 -> 自动找客户 -> 线索审核 -> 客户池 CRM
-> 独立站询盘 -> 智能开发信/WhatsApp -> 今日跟进
-> CIF 报价 -> 售后与风险 -> KPI 复盘
```

左侧导航当前顺序：

1. 市场选择
2. 自动找客户
3. 线索审核
4. 客户池 CRM
5. 独立站线索
6. 智能开发信
7. 今日跟进
8. CIF 报价
9. 售后与风险
10. KPI 看板

## 三、市场选择

入口：`01 市场选择`

用途：确定负责市场、重点城市、客户类型和搜索方向。

操作：

1. 选择目标国家。
2. 查看重点城市和市场说明。
3. 点击市场卡片，系统会把国家带入“自动找客户”。

市场分为国外和国内：

- 国外：UAE、Saudi Arabia、Kazakhstan、Russia、Qatar、Kuwait、Uzbekistan、Azerbaijan、Nigeria、Ghana、Algeria、Cote d'Ivoire、Egypt、Kyrgyzstan、Ethiopia、Oman、Armenia 等。
- 国内：按华北、华东、华南、华中、西南、西北、东北等区域搜索。

国内搜索仍处于测试阶段，数据线索可能较少或不准确。

## 四、自动找客户

入口：`02 自动找客户`

### 1. 填写条件

- 中文目标描述：说明需要寻找的客户，例如汽车进口商、经销商、展厅、车队或采购负责人。
- 目标国家（国外）：国外搜索时生效。
- 目标区域（国内）：国内搜索时生效。
- 客户来源：选择一个具体来源或综合搜索。
- 账号对象：公司账号、个人决策人或两者。
- 线索时效：不限时间、最近 3 天、7 天或 30 天。
- 搜索强度：标准或深度。

### 2. 当前可用来源

- 综合搜索（地图 + 官网）
- Google Maps 企业数据
- OpenStreetMap 地图
- 车商官网 / 汽车行业目录
- Instagram 公开账号
- Facebook 公开主页
- TikTok 公开账号
- YouTube 公开频道
- LinkedIn 公司 / 个人主页

系统已删除“社媒综合”选项。需要搜索社媒时，请选择具体平台，便于判断成本、数量和来源质量。

### 3. 执行搜索

1. 检查国家、来源、时效和目标描述。
2. 点击 `一键获客到待审核`。
3. 页面显示搜索、提取、核验和保存进度。
4. 搜索完成后自动导入线索审核，不需要再次手动导入。
5. 搜索记录会显示发现或导入数量。

系统最多同时运行 3 个获客任务。超过限制时会弹窗提示。

### 4. 搜索数量

- 目标是每次保留约 15-25 条真实、地区匹配、汽车相关的线索。
- 数量不足时，系统会扩展重点城市和商业关键词。
- 最终数量取决于平台公开数据、API 状态、去重、地区过滤和质量规则。
- 不会为了达到数量目标而保留非汽车、错误国家或明显个人娱乐账号。

### 5. 定时抓取与云端任务

- 可设置每小时、每 6 小时、每 12 小时、每天或每周执行。
- 页面关闭后，云端任务仍可继续。
- 任务支持刷新、取消、重试和删除。
- 完成结果会自动导入，搜索记录中显示导入数量。

## 五、线索准入规则

自动搜索结果必须同时重视：

- 目标国家或城市匹配。
- 汽车商业身份明确。
- 公司、展厅、进口商、经销商、分销商、车队或采购角色明确。
- 官网、社媒主页、地图地点或公开联系方式可回查。
- 食品、媒体、旅游、招聘、个人娱乐、KFC 等非汽车线索必须排除。
- 同一官网不能大量重复进入。
- 已拒绝线索默认不会再次进入新搜索结果。
- 已删除线索会保留删除指纹，避免重新导入。

### 主页邮箱规则

- 邮箱明确出现在主要线索主页中，可以作为联系邮箱。
- 这类邮箱显示为 `线索主页来源`。
- 不要求再次打开同一主页确认后才使用。
- 只存在于搜索摘要、第三方猜测或不明页面的邮箱，不自动作为联系邮箱。

## 六、线索审核

入口：`03 线索审核`

### 1. 审核状态

- 待审核
- 已审核客户
- 已拒绝

已拒绝线索可选择固定拒绝原因或填写自定义原因，也可以退回未审核。

### 2. 自动搜索线索

自动搜索线索通常显示：

- 客户名称、国家和城市
- 客户官网
- 邮箱、电话、WhatsApp 和联系人
- 来源数量、官方来源和独立域名
- 区域核实及依据
- AI 复核结论
- AI 客户画像
- 商业机会评分和分项得分

评分和 AI 只用于辅助判断，不能代替人工查看线索原文。

### 3. 线索详情

点击左侧线索后，右侧显示该线索详情。左侧列表会保持当前滚动位置，不会自动跳回第一条。

可执行：

- 查看线索原文
- 通过
- 拒绝
- 编辑
- 查看全部来源与核验详情
- 人工校准评分

### 4. 四类线索的处理差异

#### 自动搜索线索

- 进入待审核。
- 执行公开信息核验。
- 可生成 AI 复核和 AI 客户画像。
- 审核通过后进入 CRM。

#### 网址解析后保存的线索

- 在 CRM 的“手动增加线索”弹窗左侧解析官网或社媒主页。
- 保存后直接进入 CRM。
- 显示绿色 `手动添加` 标签。
- 因为使用了公开网址解析，保留来源核验和 AI 客户画像。

#### 完全手工填写的线索

- 不使用左侧解析，只填写右侧表单。
- 保存后直接进入 CRM，不进入待审核。
- 显示绿色 `手动添加` 标签。
- 不执行公开信息核验。
- 不生成 AI 复核和 AI 客户画像。

#### 独立站询盘

- 客户通过 `www.yiming-auto.com` 表单主动提交。
- 从“独立站线索”导入线索审核。
- 显示蓝色 `独立站来源` 标签。
- 按客户主动填写的信息展示，不执行公开信息核验。
- 不生成 AI 复核和 AI 客户画像。
- 批量全网补全会自动跳过独立站询盘。

## 七、当前评分标准

统一采用 100 分商业机会模型：

- 汽车业务：20 分
- 地区匹配：20 分
- 联系方式：15 分
- 官网可信：10 分
- 中国新能源：10 分
- 华为系列：10 分
- 进口分销：8 分
- 经营活跃：4 分
- 决策人：3 分

补充规则：

- 包含华为系、鸿蒙智行或对应车型信号时，同时视为中国新能源相关。
- 邮箱、电话、WhatsApp、联系人越完整，联系方式得分越高。
- 官网可打开且与车辆业务相关，官网可信度才加分。
- 明确进口、分销、授权经销、车队采购或批发能力会提高优先级。
- 地区必须有当地地址、电话区号、地图地点、官网或 AI 明确证据支持。
- 评分可人工 `-5`、`+5` 或恢复系统分。

## 八、客户池 CRM

入口：`04 客户池 CRM`

### 1. 客户列表

支持查看：

- 全部客户
- 今日待跟进
- 已逾期
- 未设下次跟进
- A/B 级优先

可按公司、国家、城市、邮箱和电话模糊搜索，并按等级、阶段和优先级筛选。

### 2. 手动增加线索

点击页面右上角 `手动增加线索`，打开画中画弹窗。

左侧“线索解析”：

1. 输入企业官网或公开社媒主页。
2. 可以输入 `example.com`、`www.example.com` 或完整 `https://...`。
3. 系统会自动补全 `https://`。
4. 点击 `解析线索`。
5. 公司、地区、联系方式、业务类型和车型会自动填入右侧表单。

右侧“手动增加线索”可填写：

- 公司名
- 国家和城市
- 客户类型
- 来源
- 适合车型
- 客户官网
- 联系人及职务
- 邮箱
- 电话
- WhatsApp
- 官网内容或备注
- 下一步动作

邮箱、电话、WhatsApp、联系人和官网均可选填。邮箱如果填写，必须是正常邮箱格式。

点击 `保存到客户池` 后直接进入 CRM。

### 3. CRM 操作

- 写开发信
- 去报价
- 修改销售阶段
- 修改下一步动作
- 设置下次跟进日期
- 退回审核
- 删除客户

### 4. 数据导出

- `导出客户表格 CSV`：导出客户池名单。
- `导出完整备份 JSON`：导出当前账号业务数据。
- `清空当前账号数据`：删除当前账号业务数据，操作前必须确认。

## 九、独立站线索

入口：`05 独立站线索`

官网表单提交后，询盘进入“官网询盘收件箱”。

操作：

1. 查看客户、联系方式、国家、车型、数量和需求。
2. 确认不是垃圾提交。
3. 点击 `导入审核`。
4. 在线索审核中确认需求。
5. 点击通过后进入客户池。

还可以使用：

- 标记处理
- 删除

独立站信息来自客户主动填写，系统不把它显示成“已核验公开信息”。

## 十、智能开发信与 WhatsApp

入口：`06 智能开发信`

1. 从客户池选择客户。
2. 带入公司、客户类型、车型和联系方式。
3. 选择 Email、WhatsApp、Instagram DM 或 LinkedIn。
4. 生成英文内容和中文含义。
5. 发送前人工检查客户名称、国家、车型和价格信息。

建议首次联系只说明：

- 我们是谁
- 可供应哪些华为系新能源汽车
- 是否愿意了解车型、配置、现车和 CIF 报价

不要在首次联系中直接发送最低价或未经确认的承诺。

## 十一、今日跟进

入口：`07 今日跟进`

建议节奏：

- Day 1：首次 Email 或 WhatsApp。
- Day 3：补充车型优势、配置、现车和交期。
- Day 7：询问是否需要经销合作、价格或批量采购方案。

每次联系后记录结果，并设置下一次跟进日期。

## 十二、CIF 报价

入口：`08 CIF 报价`

填写车型、数量、车辆单价、国内费用、海运费、保险、目的港、交期和有效期，然后计算 CIF。

注意：

- CIF 通常不包含目的国关税、VAT、认证、上牌和目的港外费用。
- 正式报价前必须核对币种、配置、数量、港口和交期。
- 保存报价会保留历史版本，不覆盖旧报价。

## 十三、售后与风险

入口：`09 售后与风险`

成交前重点确认：

- 当地进口与认证要求
- 语言、车机、地图、网络和远程 App
- 智驾功能在目标国家是否可用
- 备件、维修和质保方案
- 海运、保险、清关和仓储
- 付款通道、汇率和制裁风险

系统提示仅用于业务检查，不构成法律、税务或清关结论。

## 十四、KPI 看板

入口：`10 KPI 看板`

用于查看待审核、已核验、正式客户、已联系、有回复和报价数量。

判断方法：

- 待审核多、正式客户少：搜索条件或线索质量需要优化。
- 客户多、已联系少：销售触达不及时。
- 已联系多、回复少：客户匹配或开发信需要调整。
- 有回复多、报价少：配置、价格和交期响应速度不足。

## 十五、用户管理

仅管理员可见。

功能：

- 添加用户
- 修改密码
- 启用或停用
- 设置管理员
- 删除用户
- 设置负责国外国家
- 查看在线状态
- 查看账号操作记录

负责国家通过弹窗勾选并确认。选择“无”后，该账号不能使用自动找客户。

## 十六、系统设置

仅管理员可见。

### 1. 来源监控

- 查看各数据源状态、调用次数、发现数和错误。
- 设置来源启停和单次上限。
- 为不同国家设置来源覆盖。

### 2. 获客策略

- 设置是否仅管理员允许一键获客。
- 普通用户被暂停时，点击后提示“接口正在调试，功能暂停”。
- 设置目标线索数量、任务超时、备用来源和并发数。

### 3. 质量规则

- 严格国家匹配
- 必须具备汽车商业身份
- 拒绝个人娱乐账号
- 必须具备官网或联系方式
- 已拒绝线索禁止再次进入
- 自动导入最低分
- AI 最低置信度
- 评分权重

### 4. AI 复核

- 开启或关闭 AI 复核。
- 设置失败处理、每日调用上限和费用上限。
- 配置快速初筛、普通复核和重点复核模型。

AI 用于对已找到的公开线索分类、复核和生成客户画像，不替代 Google、地图、Apify 或社媒数据源进行实时网页搜索。

### 5. API 配置

- 管理 Google Maps、YouTube、SerpApi、Apify、DeepSeek 等 API。
- 查看工作台调用记录和供应商官方额度。
- Apify 可显示账户、月度额度、当前账期消耗、剩余额度和账期结束时间。
- 修改 API Key 并保存后，刷新官方用量会读取新绑定账号的数据。

### 6. 数据与任务

- 终止运行任务
- 清理失败任务
- 导出或恢复业务备份
- 清理过期任务和搜索记录
- 重建重复检测
- 清理删除指纹或拒绝记忆

危险操作会影响去重和历史记忆，执行前必须先导出备份。

### 7. 安全与审计

- 设置登录有效期和密码最小长度。
- 强制所有账号重新登录。
- 查看管理员操作日志。
- 重启工作台服务。

## 十七、常见问题

### 1. 修改后页面没有变化

先按：

```text
Ctrl + F5
```

强制刷新页面。长时间打开的旧标签页可能仍使用旧版 JavaScript。

### 2. 网址提示“请输入网址”

现在支持直接输入：

```text
example.com
www.example.com
https://example.com
```

系统会自动补全协议。输入内容不能包含空格。

### 3. 邮箱没有被采用

- 主线索主页公开显示的邮箱会标记为“线索主页来源”。
- 搜索摘要、Hunter 候选或无法对应主主页的邮箱不会自动采用。
- 手工表单中的邮箱可选填，但填写后必须符合邮箱格式。

### 4. 搜索结果少

- 改用深度搜索。
- 增加重点城市和汽车商业关键词。
- 分别尝试 Google Maps、TikTok、Instagram、Facebook、YouTube 和 LinkedIn。
- 放宽时效为“不限时间”。
- 检查系统设置中的来源状态、API 额度和单次上限。

### 5. 搜索结果出现错误国家

- 检查目标国家是否正确。
- 在系统设置开启“目标国家必须匹配”。
- 查看线索的区域核实和区域依据。
- 地区冲突线索应拒绝，不要仅根据名称通过。

### 6. 任务长时间停在核验阶段

- 先查看云端任务状态。
- 刷新任务，不要重复连续创建相同任务。
- 检查 Apify、Google、YouTube 或搜索 API 是否超时或额度不足。
- 管理员可在系统设置的“数据与任务”终止异常任务。

### 7. 数据没有同步

- 查看页面同步状态。
- 刷新页面重新读取云端数据。
- 确认登录的是正确账号。
- 管理员检查服务器和数据库状态。
- 操作前可先导出完整 JSON 备份。

## 十八、每日建议流程

1. 查看今日跟进和逾期客户。
2. 处理独立站新询盘。
3. 审核自动搜索的新线索。
4. 对 A/B 级客户完成首次触达。
5. 对有回复客户准备配置和 CIF 报价。
6. 记录联系结果和下次跟进日期。
7. 管理员定期查看 API 额度、失败任务和业务备份。

## 十九、使用原则

- 线索必须真实、汽车相关并符合目标地区。
- 不编造客户身份、联系方式、进口资质或采购意向。
